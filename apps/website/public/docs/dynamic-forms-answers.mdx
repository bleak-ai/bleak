# Send Answers

Collect user responses and send them back to BleakAI to get the final AI-generated result.

## Collecting Answers

After users fill out the generated questions, collect their responses in the required format:

```typescript
const answers = response.questions.map((question) => ({
  question_id: question.question_id,
  answer: getUserAnswer(question.question_id) // Your logic to get user input
}));
```

## Sending Answers

Use the `answer` method to send collected responses back to BleakAI:

```typescript
const finalResponse = await client.answer(answers);

if (finalResponse.type === "answer") {
  console.log("AI Response:", finalResponse.content);
  // Display the result in your UI
}
```

## Complete Flow

Here's how the entire conversation process works together:

```typescript
// 1. Initialize client
const client = createSimpleChatClient({
  baseUrl: "http://localhost:8000",
  apiKey: "your-bleakai-api-key",
  openaiKey: "your-openai-api-key"
});

// 2. Setup components
const {resolve} = createResolverFromConfig(yourComponentConfig);

// 3. Start conversation
let response = await client.ask("Plan my vacation");

// 4. Handle questions
if (response.type === "questions") {
  // Render questions and collect answers
  const answers = await collectUserAnswers(response.questions);

  // Send answers back
  response = await client.answer(answers);
}

// 5. Display result
if (response.type === "answer") {
  displayResult(response.content);
}
```

The conversation continues until BleakAI has enough information to provide a complete response.
