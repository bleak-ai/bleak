# Getting Started

Install and configure BleakAI to start building dynamic forms.

## Installation

```bash
npm install bleakai
```

## API Keys

Get your API keys:

1. **BleakAI API Key**: Log into the BleakAI platform dashboard
2. **OpenAI API Key**: Create one at [OpenAI's website](https://platform.openai.com/)

## Setup

```typescript
import {createSimpleChatClient} from "bleakai";

const client = createSimpleChatClient({
  baseUrl: "http://localhost:8000",
  apiKey: "your-bleakai-api-key-here",
  openaiKey: "your-openai-api-key-here"
});
```
