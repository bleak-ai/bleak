# API Reference

Complete reference for all BleakAI functions, methods, and types.

## Overview

The BleakAI SDK provides a simple interface for creating conversational AI applications. Use these methods to initialize clients, handle conversations, and integrate with your existing components.

## Documentation Sections

- **[SDK Methods](api-reference-sdk)** - Core SDK functions for creating clients and resolvers
- **[Core Methods](api-reference-core)** - BleakChat instance methods for conversations

## Quick Reference

### Client Creation

```typescript
import {createSimpleChatClient} from "bleakai";

const client = createSimpleChatClient({
  baseUrl: "http://localhost:8000",
  apiKey: "your-bleakai-api-key",
  openaiKey: "your-openai-api-key"
});
```

### Conversation Flow

```typescript
// Start conversation
const response = await client.ask("Help me plan something");

// Handle questions
if (response.type === "questions") {
  const answers = collectAnswers(response.questions);
  const finalResponse = await client.answer(answers);
}
```

### Component Mapping

```typescript
import {createResolverFromConfig} from "bleakai";

const {resolve} = createResolverFromConfig({
  text: {component: MyTextInput, description: "Text input"},
  radio: {component: MyRadioGroup, description: "Single choice"}
});
```

## Response Types

All chat methods return a `ChatResponse` with one of these types:

- **`questions`** - AI needs more information
- **`answer`** - Final response ready
- **`complete`** - Conversation finished

## Error Handling

```typescript
try {
  const response = await client.ask("Hello");
} catch (error) {
  if (error instanceof RateLimitError) {
    console.log(`Rate limited. Retry after: ${error.retryAfter}s`);
  } else if (error instanceof AuthenticationError) {
    console.log("Check your API keys");
  }
}
```
