# Initialize Chat Client

Create a chat client to communicate with BleakAI.

## Creating the Client

```typescript
import {createSimpleChatClient} from "bleakai";

const client = createSimpleChatClient({
  baseUrl: "http://localhost:8000",
  apiKey: "your-bleakai-api-key",
  openaiKey: "your-openai-api-key"
});
```

## Configuration Options

```typescript
interface ClientConfig {
  baseUrl: string; // BleakAI backend URL
  apiKey: string; // Your BleakAI API key
  openaiKey: string; // Your OpenAI API key
  timeout?: number; // Request timeout (default: 30000ms)
}
```
