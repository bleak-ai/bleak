# Initialize Chat Client

Create a chat client to communicate with the BleakAI backend.

## Creating the Client

The client handles all communication with BleakAI's conversational AI backend.

```typescript
import {createSimpleChatClient} from "bleakai";

const client = createSimpleChatClient({
  baseUrl: "http://localhost:8000",
  apiKey: "your-bleakai-api-key",
  openaiKey: "your-openai-api-key"
});
```

## Configuration Options

Customize client behavior with these options:

```typescript
interface ClientConfig {
  baseUrl: string; // BleakAI backend URL
  apiKey: string; // Your BleakAI API key
  openaiKey: string; // Your OpenAI API key
  timeout?: number; // Request timeout (default: 30000ms)
}
```

The client is now ready to handle conversations and generate dynamic questions based on user input.
